## 每日实践

<div class="daily-practice" markdown="1">
**早晨** &mdash; 十分钟静默。问：「今天我在哪里可以减少痛苦？」

**中午** &mdash; 与另一个人进行一次诚实的交流。

**傍晚** &mdash; 回顾行动。承认伤害。计划修复。

**夜间** &mdash; 放下比较。祝福众生。在没有数字噪音中休息。
</div>
